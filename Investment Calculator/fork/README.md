# Investment Calculator - Лабораторна робота №7

## Опис проекту
Інвестиційний калькулятор з графічним інтерфейсом для розрахунку складних відсотків та візуалізації результатів.

## Внесені зміни

### 1. Виправлення помилок
- Виправлено помилку в `pom.xml` (рядок 13: тег `<n>` замінено на `<n>`)

### 2. Додані функції
- **Збереження результатів у текстовий файл** - новий клас `TextFileSaver.java`
- **Кнопка "Save to Text File"** в інтерфейсі користувача
- **Apache Commons IO** - нова залежність для роботи з файлами

### 3. Нові тести
Додано 5 нових модульних тестів у класі `AdditionalInvestmentTests.java`:
- Тест збереження результатів у файл
- Тест квартального нарахування відсотків
- Тест щотижневого нарахування з внесками
- Тест з нульовою процентною ставкою
- Тест розрахунків з великими сумами

## Вимоги до системи
- Java 11 або вище
- Maven 3.6 або вище

## Інструкції зі збірки та запуску

### 1. Клонування репозиторію
```bash
git clone [URL репозиторію]
cd investment-calculator-updated
```

### 2. Збірка проекту
```bash
# Очищення попередньої збірки
mvn clean

# Компіляція
mvn compile

# Запуск тестів
mvn test

# Створення виконуваного JAR
mvn package
```

### 3. Запуск програми
```bash
# Запуск через Maven
mvn exec:java

# Або запуск JAR файлу
java -jar target/investment-calculator-1.0.0.jar
```

## Використання програми

### Основні функції:
1. **Розрахунок інвестицій** - введіть параметри та натисніть "Calculate Investment"
2. **Збереження результатів** - натисніть "Save to Text File" після розрахунку
3. **Експорт у CSV** - доступний експорт щомісячних та щорічних даних
4. **Візуалізація** - графіки росту інвестицій та розподілу доходів

### Нова функція збереження:
- Після розрахунку натисніть зелену кнопку "Save to Text File"
- Виберіть місце збереження файлу
- Файл буде збережено у текстовому форматі з усіма деталями розрахунку

## Тестування

Запуск всіх тестів:
```bash
mvn test
```

Запуск конкретного тесту:
```bash
mvn test -Dtest=AdditionalInvestmentTests
```
